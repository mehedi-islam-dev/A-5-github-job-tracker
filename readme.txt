1. What is the difference between var, let, and const?
Ans: 
    var is function scoped, hoisted, re-declarable
    const and let are block scoped,
    const used for fixed value where let allows to change values

2. What is the spread operator (...)?
Ans:
    spread operator is a feature that expands an array, objects or strings
    to individual element

3. What is the difference between map(), filter(), and forEach()?
Ans:
     map() --> transforms each element in a new array of the same length
     filter() --> finds the element based on a certain condition and provides that array
     forEach() --> perform an operation on each element but doesn't return an array.

4. What is an arrow function?
Ans:
    An arrow function is an ES6 syntax to write a function, 
    characterized by =>
    Structure: (parameter) => {body}

5. What are template literals?
Ans: A feature that use backticks (`) instead of quotes to define string 
Also used to eliminate errors when a string has several string quotes.
